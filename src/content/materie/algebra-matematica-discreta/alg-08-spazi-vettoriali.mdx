# Spazi Vettoriali

Sia $\mathbb{K}$ uno tra $\mathbb{R}$ e $\mathbb{C}$ (campo degli scalari).

## Definizione

Uno **spazio vettoriale** su $\mathbb{K}$ (detto **reale** se $\mathbb{K}=\mathbb{R}$, **complesso** se $\mathbb{K}=\mathbb{C}$) è un insieme non-vuoto $V$ i cui elementi sono detti **vettori**, dotato di due operazioni:

| Operazione | Simbolo | Tipo |
|-----------|---------|------|
| **Somma di vettori** | $+: V \times V \to V,\quad (\underline{v},\underline{w}) \mapsto \underline{v}+\underline{w}$ | interna |
| **Prodotto scalare per vettore** | $\cdot: \mathbb{K} \times V \to V,\quad (\alpha,\underline{v}) \mapsto \alpha\cdot\underline{v}$ | esterna |

che soddisfano i seguenti **8 assiomi**:

| # | Assioma | Nota |
|---|---------|------|
| 1 | $\underline{u}+(\underline{v}+\underline{w}) = (\underline{u}+\underline{v})+\underline{w}$ | $+$ associativa |
| 2 | $\underline{u}+\underline{v} = \underline{v}+\underline{u}$ | $+$ commutativa |
| 3 | $\alpha\cdot(\beta\cdot\underline{v}) = (\alpha\cdot_{\mathbb{K}}\beta)\cdot\underline{v}$ | compatibilità prodotti |
| 4 | $1\cdot\underline{v} = \underline{v}$ | identità scalare |
| 5 | $(\alpha+_{\mathbb{K}}\beta)\cdot\underline{v} = \alpha\cdot\underline{v}+\beta\cdot\underline{v}$ | distributiva (scalari) |
| 6 | $\alpha\cdot(\underline{u}+\underline{v}) = \alpha\cdot\underline{u}+\alpha\cdot\underline{v}$ | distributiva (vettori) |
| 7 | $\exists\,\underline{0}\in V$ tale che $\underline{v}+\underline{0}=\underline{v}\quad\forall\,\underline{v}$ | elemento neutro (unico) |
| 8 | $\forall\,\underline{v}\in V\;\exists\,\underline{w}\in V$ tale che $\underline{v}+\underline{w}=\underline{0}$ | opposto $-\underline{v}$ (unico) |

---

## Proprietà Derivate

Per ogni $\alpha\in\mathbb{K}$ e $\underline{v}\in V$:

| | Proprietà |
|-|-----------|
| **a)** | $\alpha\cdot\underline{0} = \underline{0}$ |
| **b)** | $0\cdot\underline{v} = \underline{0}$ |
| **c)** | $\alpha\cdot\underline{v}=\underline{0} \;\Longrightarrow\; \alpha=0$ oppure $\underline{v}=\underline{0}$ &ensp; **(legge di cancellazione)** |
| **d)** | $-(\alpha\cdot\underline{v}) = (-\alpha)\cdot\underline{v} = \alpha\cdot(-\underline{v})$ |

> **Dimostrazione di c):** se $\alpha\neq 0$, allora $\underline{v} \stackrel{4}{=} 1\cdot\underline{v} = (\alpha^{-1}\alpha)\cdot\underline{v} \stackrel{3}{=} \alpha^{-1}(\alpha\cdot\underline{v}) = \alpha^{-1}\underline{0} \stackrel{a}{=} \underline{0}$.

## Esempi Principali

### 1) Vettori colonna

$$\mathbb{R}^n = \left\{\underline{u}=\begin{pmatrix}u_1\\\vdots\\u_n\end{pmatrix} \;\middle|\; u_i\in\mathbb{R}\right\}, \qquad \mathbb{C}^n \quad \text{(analogamente)}$$

- $\underline{0} = (0,\ldots,0)^T$
- $-\underline{u} = (-u_1,\ldots,-u_n)^T$

### 2) Vettori riga

$$\mathbb{R}_n = \{(u_1,\ldots,u_n) \mid u_i\in\mathbb{R}\}, \qquad \mathbb{C}_n$$

### 3) Matrici

$$M_{m,n}(\mathbb{K})$$

- $\underline{0} = \mathbb{O}$ (matrice nulla)
- $-A = (-a_{ij})$

### 4) Polinomi

$$\mathbb{K}[x] = \{a_0+a_1x+\cdots+a_nx^n \mid n\in\mathbb{N},\, a_i\in\mathbb{K}\}$$

Sottospazio rilevante: **polinomi di grado al più $n$**

$$\mathbb{K}_n[x] = \{f(x)\in\mathbb{K}[x] \mid \deg f \leq n\}$$

Le operazioni sono ben definite su $\mathbb{K}_n[x]$ perché:
- $\deg(f+g) \leq \max\{\deg f, \deg g\} \leq n$
- $\deg(\alpha f) = \deg f \leq n$ per $\alpha\neq 0$; $\deg(0\cdot f) = -\infty \leq n$

In entrambi i casi: $\underline{0}$ = polinomio nullo $f(x)=0$, e $-f(x) = -a_0-a_1x-\cdots-a_nx^n$.

### 5) Funzioni da $[a,b]$ in $\mathbb{R}$

Siano $a<b\in\mathbb{R}$. L'insieme $F([a,b]) = \{f:[a,b]\to\mathbb{R}\}$ è uno spazio vettoriale con:
- $(f+g)(x) = f(x)+g(x)$
- $(\alpha\cdot f)(x) = \alpha\cdot f(x)$
- $\underline{0}$: funzione nulla $f(x)=0\;\forall x$
- $(-f)(x) = -f(x)$

### 6) Spazio vettoriale nullo

$$V = \{\underline{0}\}, \qquad \underline{0}+\underline{0}=\underline{0}, \quad \alpha\cdot\underline{0}=\underline{0}$$

È il più piccolo spazio vettoriale possibile.

## Sottospazi Vettoriali

**Definizione.** Sia $V$ uno spazio vettoriale su $\mathbb{K}$. Un sottoinsieme $U\subseteq V$ è un **sottospazio vettoriale** di $V$ (scritto $U\leq V$) se:

1. $\underline{0}\in U$
2. $\underline{u}_1+\underline{u}_2\in U \quad \forall\,\underline{u}_1,\underline{u}_2\in U \qquad$ ($U$ chiuso per la somma)
3. $\alpha\cdot\underline{u}\in U \quad \forall\,\underline{u}\in U,\,\forall\,\alpha\in\mathbb{K} \qquad$ ($U$ chiuso per il prodotto scalare)

> **Nota:** $U\subseteq V$ = sottoinsieme; $U\leq V$ = sottospazio. Se $U\leq V$, allora $U$ è a sua volta spazio vettoriale su $\mathbb{K}$ (con le operazioni ristrette da $V$).

> **Attenzione:** la condizione 1 ($\underline{0}\in U$) è necessaria — un sottoinsieme che non contiene $\underline{0}$ non può essere un sottospazio.

### Esempi

**1) Sottospazio di $\mathbb{R}^3$**

$$U = \left\{\begin{pmatrix}a\\0\\b\end{pmatrix} : a,b\in\mathbb{R}\right\} \leq \mathbb{R}^3$$

Verifica:
1. $\underline{0} = \begin{pmatrix}0\\0\\0\end{pmatrix}\in U$ (scegliendo $a=b=0$). ✓
2. $\begin{pmatrix}a_1\\0\\b_1\end{pmatrix}+\begin{pmatrix}a_2\\0\\b_2\end{pmatrix} = \begin{pmatrix}a_1+a_2\\0\\b_1+b_2\end{pmatrix}\in U$. ✓
3. $\alpha\cdot\begin{pmatrix}a\\0\\b\end{pmatrix} = \begin{pmatrix}\alpha a\\0\\\alpha b\end{pmatrix}\in U$. ✓

---

**2) Spazio nullo di una matrice**

Sia $A\in M_{m,n}(\mathbb{C})$. Il suo **spazio nullo** è:

$$\boxed{N(A) = \{\underline{v}\in\mathbb{C}^n \mid A\cdot\underline{v}=\underline{0}\} \leq \mathbb{C}^n}$$

$N(A)$ è l'insieme delle soluzioni del sistema lineare omogeneo $A\cdot\underline{x}=\underline{0}$.

Verifica che $N(A)\leq\mathbb{C}^n$:
1. $A\cdot\underline{0}=\underline{0}$ → $\underline{0}\in N(A)$. ✓
2. $A(\underline{u}_1+\underline{u}_2) = A\underline{u}_1+A\underline{u}_2 = \underline{0}+\underline{0}=\underline{0}$. ✓
3. $A(\alpha\cdot\underline{u}) = \alpha\cdot(A\underline{u}) = \alpha\cdot\underline{0}=\underline{0}$. ✓

---

**3) Sottospazi di $M_n(\mathbb{C})$**

| Insieme | Definizione | Nome |
|---------|------------|------|
| $W_1$ | $\{A\in M_n(\mathbb{C}) \mid A^T = -A\}$ | matrici **antisimmetriche** di ordine $n$ |
| $W_2$ | $\{A\in M_n(\mathbb{C}) \mid A^H = A\}$ | matrici **hermitiane** di ordine $n$ |

Verificare se $W_1, W_2\leq M_n(\mathbb{C})$ è un esercizio.

---

### Criterio alternativo

La condizione $\underline{0}\in U$ può essere sostituita da $U\neq\emptyset$ (dato che valgono già chiusura per somma e per prodotto scalare):

> $U\neq\emptyset$ + chiusura per prod. scalare $\Rightarrow \underline{0}\in U$, poiché $\exists\,\underline{u}\in U$ e $0\cdot\underline{u}=\underline{0}\in U$.

### Proprietà dei sottospazi

Sia $V$ sp. vett. su $\mathbb{K}$. Allora:

1. **Transitività:** $U\leq W\leq V \Rightarrow U\leq V$.
2. **Intersezione:** se $U_1\leq V$ e $U_2\leq V$ allora $U_1\cap U_2\leq V$.
3. **Sottospazi banali:** $\{\underline{0}\}\leq V$ (sottospazio nullo) e $V\leq V$.

### Somma di sottospazi

Siano $U_1,U_2\leq V$. La **somma** è:

$$\boxed{U_1+U_2 = \{\underline{u}_1+\underline{u}_2 \mid \underline{u}_1\in U_1,\;\underline{u}_2\in U_2\}\leq V}$$

Se $U_1\cap U_2=\{\underline{0}\}$, la somma si dice **diretta** e si scrive $U_1\oplus U_2$.

**Esempio** ($V=\mathbb{C}^3$):

$$U_1=\left\{\begin{pmatrix}a\\0\\b\end{pmatrix}:a,b\in\mathbb{C}\right\},\quad U_2=\left\{\begin{pmatrix}0\\h\\0\end{pmatrix}:h\in\mathbb{C}\right\}$$

$U_1\cap U_2=\{\underline{0}\}$, quindi $U_1\oplus U_2 = \mathbb{C}^3$.

### Insieme dei multipli

Sia $\underline{v}\in V$. L'**insieme dei multipli** di $\underline{v}$ è:

$$\boxed{\langle\underline{v}\rangle = \{\alpha\cdot\underline{v}\mid\alpha\in\mathbb{K}\}\leq V}$$

- Se $\underline{v}=\underline{0}$: $\langle\underline{0}\rangle=\{\underline{0}\}$.
- Se $\underline{v}\neq\underline{0}$: $\langle\underline{v}\rangle$ ha infiniti elementi (la mappa $\alpha\mapsto\alpha\cdot\underline{v}$ è iniettiva).

## 4.2 Combinazioni Lineari e Insiemi di Generatori

**Definizione.** La **combinazione lineare** dei vettori $\underline{v}_1,\ldots,\underline{v}_n\in V$ con coefficienti $\alpha_1,\ldots,\alpha_n\in\mathbb{K}$ è il vettore:
$$\underline{v} = \alpha_1\underline{v}_1+\cdots+\alpha_n\underline{v}_n$$

> Uno stesso vettore può ammettere più rappresentazioni come combinazione lineare degli stessi vettori (se questi sono L.D.).

**Definizione.** Il **sottospazio generato** da $\underline{v}_1,\ldots,\underline{v}_n$ è:

$$\boxed{\langle\underline{v}_1,\ldots,\underline{v}_n\rangle = \{\alpha_1\underline{v}_1+\cdots+\alpha_n\underline{v}_n\mid\alpha_i\in\mathbb{K}\}\leq V}$$

È il **più piccolo** sottospazio di $V$ contenente $\underline{v}_1,\ldots,\underline{v}_n$. Per convenzione $\langle\emptyset\rangle=\{\underline{0}\}$.

**Definizione.** Un insieme $\mathcal{S}=\{\underline{v}_1,\ldots,\underline{v}_n\}$ è un **insieme di generatori** di $V$ se $\langle\mathcal{S}\rangle=V$.

### Indipendenza lineare

L'insieme $\mathcal{S}=\{\underline{v}_1,\ldots,\underline{v}_n\}$ è:

- **Linearmente indipendente (L.I.)** se:
$$\alpha_1\underline{v}_1+\cdots+\alpha_n\underline{v}_n=\underline{0}\;\Longrightarrow\;\alpha_1=\cdots=\alpha_n=0$$

- **Linearmente dipendente (L.D.)** se esistono $\alpha_1,\ldots,\alpha_n$ **non tutti nulli** tali che $\alpha_1\underline{v}_1+\cdots+\alpha_n\underline{v}_n=\underline{0}$.

> L.D. significa che almeno un vettore è combinazione lineare degli altri.

### Proprietà degli insiemi L.I.

1. $\{\underline{v}\}$ è L.I. $\iff$ $\underline{v}\neq\underline{0}$.
2. Ogni sottoinsieme di un insieme L.I. è L.I.
3. Un insieme L.I. non contiene vettori ripetuti né un vettore e un suo multiplo.

### Esempi canonici

| Spazio $V$ | Insieme $\mathcal{S}$ | Generatori? | L.I.? |
|---|---|---|---|
| $\mathbb{K}^n$ | $\mathcal{E}=\{\underline{e}_1,\ldots,\underline{e}_n\}$ (colonne di $\mathbb{1}_n$) | ✓ | ✓ |
| $\mathbb{K}_n[x]$ | $\{1,x,x^2,\ldots,x^n\}$ | ✓ | ✓ |
| $M_{m,n}(\mathbb{K})$ | $\{E_{ij}\mid 1\leq i\leq m,\,1\leq j\leq n\}$ | ✓ | ✓ |
| $\mathbb{K}[x]$ | $\{1,x,x^2,\ldots\}$ | ✓ | ✓ |

> $\mathbb{K}[x]$ **non** ha insiemi di generatori finiti: se esistesse $W=\{f_1,\ldots,f_m\}$ con grado massimo $n$, allora $x^{n+1}$ non sarebbe combinazione lineare degli $f_i$. Contraddizione.

**D'ora in poi** si considerano solo spazi **finitamente generati** (esiste un insieme di generatori finito).

## 4.3 Basi e Dimensione

**Definizione.** Una **base** di $V$ è un insieme di vettori che sia simultaneamente:

1. insieme di generatori di $V$
2. L.I.

**Esempio.** $\mathcal{E}=\{\underline{e}_1,\ldots,\underline{e}_n\}$ è la **base canonica** di $\mathbb{K}^n$.

> Uno spazio vettoriale può avere più basi (es. $\mathbb{R}^2$ ha base canonica ma anche $\left\{\begin{pmatrix}1\\1\end{pmatrix},\begin{pmatrix}1\\-1\end{pmatrix}\right\}$).

### Teorema di Esistenza ed Equipotenza delle Basi

Sia $V$ sp. vett. su $\mathbb{K}$. Allora:

1. Ogni insieme di generatori di $V$ **contiene almeno una base**.
2. Tutte le basi di $V$ hanno lo stesso numero di vettori, detto **dimensione** di $V$:
$$\dim V$$

**Dimensioni dei casi principali:**

| Spazio | Dimensione |
|--------|-----------|
| $\mathbb{K}^n$ | $n$ |
| $\mathbb{K}_n[x]$ | $n+1$ |
| $M_{m,n}(\mathbb{K})$ | $m\cdot n$ |
| $\{\underline{0}\}$ | $0$ |

**Conseguenza.** Se $\dim V=m$ e $\mathcal{S}$ è un insieme di generatori di $V$ con esattamente $m$ elementi, allora $\mathcal{S}$ è una base.

> *Dimostrazione:* il Teorema garantisce una base $\mathcal{B}\subseteq\mathcal{S}$; poiché $|\mathcal{B}|=m=|\mathcal{S}|$, si ha $\mathcal{B}=\mathcal{S}$.

### Proprietà delle dimensioni

Sia $V$ sp. vett. su $\mathbb{K}$ e $U,U_1,U_2\leq V$. Allora:

1. $\dim U\leq\dim V$; inoltre $\dim U=\dim V\iff U=V$ e $\dim U=0\iff U=\{\underline{0}\}$.
2. **Formula di Grassmann:**
$$\dim(U_1+U_2) = \dim U_1+\dim U_2-\dim(U_1\cap U_2)$$

## Base Ordinata e Coordinate

**Definizione.** Una **base ordinata** di $V$ è una base in cui l'ordine degli elementi è fissato.

**Proprietà.** Se $\mathcal{B}=(\underline{v}_1,\ldots,\underline{v}_n)$ è una base ordinata di $V$, allora ogni $\underline{v}\in V$ si scrive **in modo unico** come:
$$\underline{v}=\alpha_1\underline{v}_1+\cdots+\alpha_n\underline{v}_n,\qquad \alpha_i\in\mathbb{K}$$

Il **vettore delle coordinate** di $\underline{v}$ rispetto a $\mathcal{B}$ è:
$$C_{\mathcal{B}}(\underline{v})=\begin{pmatrix}\alpha_1\\\vdots\\\alpha_n\end{pmatrix}\in\mathbb{K}^n$$

**Esempio.** $V=\left\{\begin{pmatrix}a&b\\0&c\end{pmatrix}:a,b,c\in\mathbb{C}\right\}$, base ordinata $\mathcal{B}=\left(\begin{pmatrix}1&0\\0&1\end{pmatrix},\begin{pmatrix}1&1\\0&0\end{pmatrix},\begin{pmatrix}0&0\\0&2\end{pmatrix}\right)$.

Per $\underline{v}=\begin{pmatrix}2&3\\0&4\end{pmatrix}$:
$$\underline{v}=(-1)\begin{pmatrix}1&0\\0&1\end{pmatrix}+3\begin{pmatrix}1&1\\0&0\end{pmatrix}+\tfrac{5}{2}\begin{pmatrix}0&0\\0&2\end{pmatrix} \;\Longrightarrow\; C_{\mathcal{B}}(\underline{v})=\begin{pmatrix}-1\\3\\5/2\end{pmatrix}\in\mathbb{C}^3$$

### Proprietà della mappa delle coordinate

Sia $V$ sp. vett. su $\mathbb{K}$, $\dim V = n$, $\mathcal{B} = (\underline{v}_1,\ldots,\underline{v}_n)$ base ordinata.

1. **Base canonica:** se $V=\mathbb{K}^n$ ed $\mathcal{E}=(\underline{e}_1,\ldots,\underline{e}_n)$ è la base canonica, allora $C_{\mathcal{E}}(\underline{v})=\underline{v}$ per ogni $\underline{v}\in V$.
2. **Vettori di base:** $C_{\mathcal{B}}(\underline{v}_i)=\underline{e}_i$ per ogni $i=1,\ldots,n$.
3. **Linearità:** la mappa $C_{\mathcal{B}}: V \to \mathbb{K}^n$ è un'applicazione lineare (vedi §4.5).

## 4.4 Spazi Vettoriali Associati a Matrici

Sia $A \in M_{m,n}(\mathbb{K})$, con colonne $\underline{c}_1,\ldots,\underline{c}_n \in \mathbb{K}^m$ e righe $r_1,\ldots,r_m$.

### Spazio delle colonne

$$\boxed{C(A) = \langle \underline{c}_1, \underline{c}_2, \ldots, \underline{c}_n \rangle \leq \mathbb{K}^m}$$

**Strategia per trovare una base di $C(A)$:**

1. Calcolare una forma ridotta $U$ di $A$.
2. $\dim C(A) = \mathrm{rk}(A)$ = n° colonne dominanti di $U$.
3. Una base di $C(A)$ è formata dalle colonne di $A$ nelle **stesse posizioni** delle colonne dominanti di $U$.

> **Attenzione:** in generale $C(A) \neq C(U)$, quindi **non** si prendono le colonne dominanti di $U$.

### Spazio delle righe

$$\boxed{R(A) = \langle \overline{r_1}^T, \overline{r_2}^T, \ldots, \overline{r_m}^T \rangle \leq \mathbb{K}^n}$$

dove $\overline{r_i}^T$ è la H-trasposta della riga $i$. Si ha $R(A) = C(A^H)$.

**Strategia per trovare una base di $R(A)$:**

1. Calcolare una forma ridotta $U$ di $A$.
2. $R(A) = R(U)$: una base è data dalle H-trasposte delle **righe non nulle** di $U$.

### Spazio nullo

$$\boxed{N(A) = \{\underline{v} \in \mathbb{K}^n \mid A\underline{v} = \underline{0}\} \leq \mathbb{K}^n}$$

$N(A)$ coincide con l'insieme delle soluzioni del sistema omogeneo $A\underline{x}=\underline{0}$. Si ha $N(A) = N(U)$ per ogni forma ridotta $U$ di $A$, e:

$$\dim N(A) = n - \mathrm{rk}(A) = \text{n° colonne libere di } U$$

**Strategia per trovare una base di $N(A)$:**

1. Calcolare $U$ e risolvere $U\underline{x}=\underline{0}$.
2. La soluzione generale dipende da $k = \dim N(A)$ parametri liberi; i $k$ vettori ottenuti ponendo un parametro $=1$ e gli altri $=0$ formano una base di $N(A)$.
3. Se tutte le colonne di $U$ sono dominanti, $N(A)=\{\underline{0}\}$ e la base è vuota.

### Riepilogo dimensioni

$$\dim C(A) = \dim R(A) = \mathrm{rk}(A), \qquad \dim N(A) = n - \mathrm{rk}(A)$$

### Metodo per stabilire se $\mathcal{S}$ è L.I. / base di $V$

Dato $\mathcal{S} = \{\underline{v}_1,\ldots,\underline{v}_k\} \subseteq V$, si forma la matrice $A$ avente i vettori di $\mathcal{S}$ come colonne:

- **$\mathcal{S}$ è L.I.** $\iff$ $\mathrm{rk}(A) = k$ (= n° vettori di $\mathcal{S}$).
- **Base contenuta in $\mathcal{S}$:** le colonne di $A$ nelle posizioni dominanti di $U$ formano una base di $\langle\mathcal{S}\rangle$.
- **$\mathcal{S}$ è base di $\mathbb{K}^n$** $\iff$ $k = n$ e $\mathrm{rk}(A) = n$ (cioè $A$ invertibile).
